# Assistia Sanity Studio

Standalone Sanity Studio for Assistia portfolio and blog content.

## Included Schemas

- `project`
- `post`

## Local Development

1. Install dependencies:
   `npm install`
2. Start Studio:
   `npm run dev`

The Studio connects to:

- Project ID: `m3497g80`
- Dataset: `production`

## Deploy

Use either:

- `npm run deploy` for Sanity-hosted Studio
- or deploy this repo to Vercel as a standalone Studio app
